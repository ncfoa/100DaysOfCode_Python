Welcome to my George Carlin Quotes API

I built this api as part of a course on Udemy where the task was to create a RESTful API. I didn't want to do the "Cafe" database that the instructor was using so I built my own. Enjoy!



GET

http://localhost:5000/quote

Make a standard GET request to the /quote endpoint to get a random quote.



POST

http://localhost:5000/new?quote=this is new quote

Make a POST request to the /new endpoint with a parameter of "quote" and your quote as the data.



PUT

http://localhost:5000/edit?id=165"e=this is an edited quote

Make a PUT request to the /edit endpoint with the id and edited quote in the parameters



Delete

http://localhost:5000/delete?id=165

Make a delete request to the /delete endpoint with the id as a parameter
